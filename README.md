# supreme_shuttle_fleet_management_system

# Programming Languages
•	Front end: HTML, CSS, and JavaScript.
•	Back end: PHP.
# Database Tools
•	MySQL.
# Graphics Tools
•	Adobe Photoshop. 
•	Adobe Illustrator.
# Server  
•	XAMPP server.
# Operating System
